# Pigskin Death Count

Pigskin Death Count is a browser-based arcade football game built with HTML5 Canvas and JavaScript.

## Play

Play the game through the GitHub Pages link:

https://mhall350.github.io/Pigskin-Deathcount/

## Controls

| Action | Key |
|---|---|
| Move | Arrow Keys or WASD |
| Pass | P |
| Tackle | Space |
| Extra Point | K |
| Dash | J |
| Start / Continue | Enter |

## Objective

Lead Seattle through five opponents and win the championship.
